Todos os códigos dessa pasta funcionam individualmente, porem o código em pyhton denominado "gráfico" serve como exibição gráfica dos pontos de calibração obtidos no código "calculo_automatizado_com_calibracao_manual_1_sensor.ino", porem o processo de plotagem não é automatizado. Para exibir corretamente o gráfico que representa a função linearizada de cada sensor deve-se rodar o código ".ino" e nele serão salvos e exibidos os valores médios dos sensores obtidos na calibração, esses valores devem ser inseridos manualmente no código python para que o mesmo plote corretamente o gráfico.
